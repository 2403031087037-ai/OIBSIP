#  Wine Quality Prediction using Machine Learning

## 📌 Overview
This project focuses on analyzing and predicting wine quality using Machine Learning techniques.  
The notebook includes complete data preprocessing, exploratory data analysis (EDA), visualization, and model building using the Random Forest algorithm.

---

## 🎯 Objectives

- Analyze wine quality data
- Identify important chemical properties affecting wine quality
- Perform exploratory data analysis (EDA)
- Build a machine learning model for prediction
- Evaluate model performance

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
