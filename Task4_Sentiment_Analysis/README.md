# 🐦 Twitter Sentiment Analysis Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard for analyzing sentiment from Twitter data. The dashboard helps users understand public opinion by categorizing tweets into different sentiment groups and visualizing trends through dynamic charts and KPIs.

The report transforms raw Twitter text data into meaningful insights using data visualization and sentiment analysis techniques.

---

## 🎯 Objectives

- Analyze sentiment from Twitter data.
- Measure positive, negative, and neutral opinions.
- Track overall social media engagement.
- Identify sentiment distribution patterns.
- Provide actionable insights through interactive visualizations.

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Sentiment Analysis Dataset

## 📊 Dashboard Features

### KPI Metrics

- Sentiment Distribution Charts

- Cross-visual interaction
