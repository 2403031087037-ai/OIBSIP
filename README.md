# OIBSIP
# **Retail Sales Dashboard**

This project provides an interactive dashboard for analyzing retail sales performance and customer insights.

## **Technologies Used**

* Power BI
* Data Visualization
* Business Intelligence

## **Dashboard Features**

* Sales performance overview
* Monthly sales trend analysis
* Product category insights
* Customer demographic analysis
* Interactive filters and KPI cards

## **Key Insights**

* Highest sales recorded in May
* Electronics is the top-performing category
* Male customers contribute the highest customer share
