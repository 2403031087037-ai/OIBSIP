# 🏠 House Price Prediction Dashboard

## 📌 Project Overview
This Power BI dashboard provides an interactive analysis of housing data and house price predictions.  
The report focuses on exploratory data analysis (EDA), feature impact, and overall model performance using visual insights.

---

## 📊 Dashboard Features

- Average House Price Analysis
- Maximum & Minimum House Price Tracking
- Median House Price Insights
- Total Houses Overview
- Feature Impact Visualization
- Preferred Area Filtering
- Interactive Slicers & Visual Exploration

---

## 🛠 Tools & Technologies

- Power BI (.pbix)
- Data Visualization
- Exploratory Data Analysis (EDA)
- Predictive Analytics
